Questions & Answers

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure and layout of content on the web.

How do you create a hyperlink in HTML?

To create a hyperlink in HTML, use the <a> (anchor) element with the href attribute to specify the URL of the link. Example: <a href="https://www.example.com">Visit Example</a>.

What is Head Tag?

The head tag head contains all the elements describing the document.

What is Title Tag?

The title tag title specifies the HTML page title, which is shown in the browser's title bar.

What is Body Tag?

The body tag body is where you insert your web page's content.

What is Paragraph Tag?

A paragraph tag p is used to define a paragraph on a web page.

What is Heading Tag?

The HTML heading tag is used to define the heading of the HTML document. The h1 tag defines the most important tag, and h6 defines the least.

What are block-level elements?

Block-level elements are HTML elements that start on a new line and occupy the full width available within their parent element. Examples include <div>, <p>, <h1> to <h6>, <ul>, <ol>, etc.

What is List Tag?

The list tag is used if you want to enter the contents in the listed order. There are two types of lists.

  • Unordered list

    What are inline elements?

    Inline elements are HTML elements that do not start on a new line and only occupy as much width as necessary to display their content. Examples include <span>, <a>, <strong>, <em>, <img>, <input>, etc.
